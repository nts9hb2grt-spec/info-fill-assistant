# 信息填充助手

关键词匹配 + 文档抽取 + Excel 回填工具

基于 Electron 构建的桌面应用程序，用于从 Word 文档中自动提取内容并填充到 Excel 表格。

## 功能特性

- **文件选择** - 支持选择 Excel 模板文件和 Word 源文件
- **关键词匹配** - 读取 Excel 第一列作为关键词
- **内容提取** - 从 Word 文档中匹配并提取相关内容
- **自动回填** - 将提取结果写入 Excel 第二列
- **预览功能** - 处理前可预览 Excel 和 Word 内容
- **进度显示** - 实时显示处理进度和统计结果

## 使用方法

### 1. 安装依赖

```bash
npm install
```

### 2. 运行开发模式

```bash
npm run dev
```

### 3. 打包发布

```bash
npm run build
```

打包后的 EXE 文件位于 `dist/` 目录。

## 操作流程

1. 点击「选择 Excel 文件」按钮，选择模板文件（第一列包含关键词/占位符）
2. 点击「选择 Word 文件」按钮，选择包含要提取内容的源文件
3. 可选：点击「预览」按钮查看文件内容
4. 点击「开始处理」按钮，系统将自动：
   - 读取 Excel 第一列作为关键词
   - 在 Word 文档中搜索包含关键词的内容
   - 将匹配结果填充到 Excel 第二列
5. 处理完成后，结果文件保存为 `原文件名_结果.xlsx`

## 技术栈

- **Electron** - 桌面应用框架
- **xlsx** - Excel 文件读写
- **mammoth** - Word 文档解析
- **electron-builder** - 应用打包
